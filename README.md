# Dribbble Landing Page Clone

A frontend clone of the Dribbble landing page built using HTML and CSS.  
This project focuses on practicing modern layout techniques, UI cloning, and responsive design.

## Features

- Clean landing page UI inspired by Dribbble
- Responsive layout
- Flexbox-based layout structure
- Modern and minimal design

## Tech Stack

- HTML
- CSS
